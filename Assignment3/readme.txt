The code is available on github at location: https://github.com/yogeshpaul/ml-gatech/tree/main/Assignment3

Code format: Jupyter notebook

1. Dependencies=python3.10, standard ML libraries like numpy, pandas, matplotlib, sklearn, seaborn.
2. Install imbalanced-learn: pip install -U imbalanced-learn.
3. Download the jupyter notebook "pa3_clustering_dr.ipynb" from above location. Code for both datasets is in same file.
4. Download datasets from above location: healthcare-dataset-stroke-data.csv and BankChurners.csv.
5. Run jupyter notebook from top to bottom without requiring any changes.
6. Plots are saved in same directory where jupyter notebook is present. To avoid saving of plots, set SAVE_IMAGES=False in 2nd cell of the notebook.
